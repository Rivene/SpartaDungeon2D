# SpartaDungeon2D
Unity 2D 게임 팀 프로젝트

## 프로젝트 소개
던전에서 몬스터를 잡고 아이템을 파밍해서 던전마지막층을 공략하는것이 목표

## 멤버 구성
팀장 이인규: 
팀원 김준서: 
팀원 석동구: 플레이어 구현, 플레이어 공격 구현, 몬스터 랜덤 스폰 구현
팀원 이서현: 

## 개발 기간
20240226 ~ 20240304

## 유니티 버전
2022.3.2f1
